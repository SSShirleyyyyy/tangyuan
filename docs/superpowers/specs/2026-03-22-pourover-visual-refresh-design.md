# Pourover Visual Refresh Design

**Goal:** Refresh the demo so it feels like a Japanese coffee tools editorial instead of a generic web dashboard.

## Direction

- Primary mood: quiet tool catalog
- Secondary mood: paper journal
- Avoid: glassmorphism, heavy gradients, dashboard cards, loud accent overload

## Visual System

- Background: warm paper and linen tones
- Text: charcoal and brown-black
- Accent: restrained clay brown
- Surfaces: bordered paper panels with light shadow only
- Typography: serif headlines with calm sans body copy

## Layout Changes

- Sidebar becomes more editorial and less app-like
- Home hero becomes a single-column masthead with one dominant CTA
- Recent brews become journal entries instead of app cards
- Brew page becomes grouped record sheets instead of a form grid
- Suggestion page becomes photo sample plus recommendation dossier

## Constraints

- Keep existing local data and navigation behavior
- Preserve responsive layout
- Do not add network dependencies
